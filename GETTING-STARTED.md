## Troubleshooting

### "Please configure your API key"
- You haven't added an API key yet
- Go to Settings and add it

### "Failed to fetch" or API errors
- Check your API key is correct
- Make sure you have credits in your Anthropic account
- Check console for error details (F12 in browser)

### "Insufficient credits"
- Add more credits at console.anthropic.com
- Â£5-10 is usually plenty

### My data disappeared
- Did you clear browser cache/cookies?
- Did you switch browsers?
- Each browser stores data separately

## Tips for Best Results

### Medication Safety
- Always fill in "Current Medications" in Settings
- The AI will check supplement interactions
- Especially important if taking Tamoxifen or other medications

### Weekly Tracking
- Don't stress about daily "fails"
- Check your weekly totals - that's what matters
- A big day followed by a small day = balanced week

### Use One Main Browser
- Pick one browser/device as your "main" tracker
- Keeps your weekly data consistent
- Makes tracking easier
