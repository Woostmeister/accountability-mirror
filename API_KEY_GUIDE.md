# API Keys: Quick Guide

## Important: Your Claude.ai Subscription vs API Access

**Your Claude.ai subscription does NOT include API access.**

- **claude.ai subscription** = Web/app interface access (what you're using to chat with me now)
- **API access** = Separate service for developers to integrate Claude into their own apps

## Getting an API Key (Two Options)

### Option 1: Claude API (Recommended for you)

**You need a separate Anthropic API account:**

1. Go to: https://console.anthropic.com/
2. Sign up (can use same email as claude.ai, but it's a separate account)
3. Go to "API Keys" in the menu
4. Click "Create Key"
5. Copy the key (starts with `sk-ant-`)

**Pricing:**
- Pay-as-you-go (no subscription needed)
- Very cheap for personal use: ~$0.003 per message
- You add credits (minimum $5)
- Example: $5 could last you months for this app

**Free tier:**
- Anthropic sometimes offers free trial credits
- Check the console for current promotions
